언어모델의 한계는 언어를 입력을 받고 출력을 언어로 하는게 다이다.

우리는 수많은 App을 사용하고 있고, 다양한 방법으로 연동을 하고 있다.
이러한 App들과 연동을 하려면 API 스펙에 맞게 개발을 해야 한다.

예를 들면 Jira 등록을 개발하려면 Jira API Spec에 맞게 서버도 만들어야 하고, 관련해서 client 앱도 만들어야 한다.
